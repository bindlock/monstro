# Reference #

## Getting started ##

### Create new project ###
```
monstro new project example
cd example
```

### Create new module ###
```
monstro new module modules/example
```

### Update settings ###
Set `secret_key` and add `example` to `modules` in `settings/base`.

### Run server ###
```
./manage.py run
```
